# ServerPulse - サーバー監視パネル / Server Monitoring Panel / 伺服器監控面板

## 日本語 (Japanese)

### 機能紹介

**🌐 多言語サポート**
- 11種類の言語に対応（简体中文、繁体中文、英語、日本語、韓国語、ドイツ語、フランス語、スペイン語、ポルトガル語、ロシア語、アラビア語）
- リアルタイムの言語切り替え、ブラウザ閉じても設定が保持されます

**🔒 プライバシー保護**
- 公共表示ページでIPアドレスやWebアドレスを非表示にできます
- 機密情報を保護しながらサーバーステータスを共有

**🎨 テーマ管理**
- 「月」テーマ（デフォルト）搭載
- .thk形式のサポートでカスタムテーマのアップロード可能
- 作者: LunarByte / バージョン: 1.0.2

**🔌 プラグインシステム**
- WebSocketリアルタイム推送プラグイン（ビルトイン）
- .plk形式でカスタムプラグインのアップロード可能
- 公式プラグインストアに対応（開発中）

**📊 サーバー監視**
- プローブプロトコル対応：CPU、GPU、RAM、SSD、帯域幅などをリアルタイム監視
- 5分間隔のハートビート記録
- 応答時間、心拍記録 Trends Chart
- 国旗アイコン表示（flagcdn.com使用）

**🔔 通知設定**
- メール通知
- Webhook通知
- サウンドアラート
-  DingTalk、企業WeChatロボット対応
- オフライン通知ディレイ設定
- 回復通知、每日サマリー、每周レポート

**📱 レスポンシブデザイン**
- モバイルからデスクトップまで最適化
- モバイル専用ドローアーメニュー

**✨ ガラス態デザイン**
- 高円角ガラス風効果
- 浅い桃色背景 + ラベンダーアクセント
- 滑らかなアニメーションとトランジション

**🌍 タイムゾーンサポート**
- 45の世界タイムゾーンに対応
- グローバルチームに最適

### 使い方

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm run dev

# 本番ビルド
pnpm build

# プレビュー
pnpm preview
```

### アクセスURL
- 管理パネル: http://localhost:5173/
- 公開表示ページ: http://localhost:5173/public

## English

### Features

**🌐 Multi-Language Support**
- Supports 11 languages (Simplified Chinese, Traditional Chinese, English, Japanese, Korean, German, French, Spanish, Portuguese, Russian, Arabic)
- Real-time language switching with persistent settings

**🔒 Privacy Protection**
- Hide IP addresses and web addresses on public display page
- Share server status while protecting sensitive information

**🎨 Theme Management**
- Built-in "Moon" theme (default)
- Support for custom .thk theme uploads
- Author: LunarByte / Version: 1.0.2

**🔌 Plugin System**
- WebSocket real-time push plugin (built-in)
- Support for custom .plk plugin uploads
- Official plugin store (in development)

**📊 Server Monitoring**
- Probe protocol support: Real-time monitoring of CPU, GPU, RAM, SSD, bandwidth
- 5-minute interval heartbeat records
- Response time and heartbeat trend charts
- Country flag icons (using flagcdn.com)

**🔔 Notification Settings**
- Email notifications
- Webhook notifications
- Sound alerts
- DingTalk, WeCom robot support
- Offline notification delay settings
- Recovery notifications, daily summaries, weekly reports

**📱 Responsive Design**
- Optimized for mobile to desktop
- Mobile drawer menu

**✨ Glass Morphism Design**
- High rounded glass effects
- Light peach background with lavender accents
- Smooth animations and transitions

**🌍 Timezone Support**
- Supports 45 global timezones
- Perfect for global teams

### Usage

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

### Access URLs
- Admin Panel: http://localhost:5173/
- Public Display Page: http://localhost:5173/public
